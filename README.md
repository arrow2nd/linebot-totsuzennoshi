## 使い方

![無題22_20200505145149](https://user-images.githubusercontent.com/44780846/81038126-f5bee100-8edf-11ea-9eb2-4f1b1549fb14.png)

## 注意
- 文字によっては表示が崩れる場合があります。
- サーバーの都合上、最初の返信に10秒程かかる場合があります。
